# GlobalFresh - Food Import & Export Portfolio Website

A modern, sleek portfolio website for a premium food import and export company.

## Features

- **Modern Design**: Clean, professional design with a contemporary color palette (forest green #2d5016 and sage green #4a7c2c)
- **Hero Video Section**: Auto-playing background video in the hero section
- **Video Integration**: HTML5 video elements for about, products, and values sections
- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: Fade-in animations and hover effects for enhanced user experience
- **Mobile Navigation**: Hamburger menu for mobile devices with smooth toggle functionality
- **Modern Typography**: Uses Playfair Display for headings and Poppins for body text
- **Professional Styling**: Gradient backgrounds, smooth transitions, and modern button designs

## Sections

1. **Hero Section**: Eye-catching introduction with auto-playing video background
2. **About**: Company information with video showcase
3. **Products**: Premium product portfolio details with video
4. **Values**: Core business values presentation with video
5. **Contact**: Professional contact form with modern styling
6. **Footer**: Clean footer with copyright information

## Video Assets Required

The website expects the following video files in the project root directory:
- `food-video.mp4` - Hero section background video
- `about-video.mp4` - About section video
- `products-video.mp4` - Products section video
- `values-video.mp4` - Values section video

Replace these with your own food-related videos for the best experience.

## Color Scheme

- **Primary Green**: #2d5016 (Dark forest green)
- **Secondary Green**: #4a7c2c (Sage green)
- **Background**: #fafaf8 (Off-white)
- **Text**: #333 (Dark gray)

## Getting Started

1. Replace the video placeholders (`food-video.mp4`, `about-video.mp4`, `products-video.mp4`, `values-video.mp4`) with your own video files
2. Update the company name "GlobalFresh" in the HTML if needed
3. Customize the content in each section to match your company
4. Open `index.html` in a web browser to view the site

## Customization

To customize the website:

1. **Colors**: Edit the CSS variables in `iDonateStyle.css` (lines with #2d5016 and #4a7c2c)
2. **Content**: Update text content in `index.html`
3. **Videos**: Replace video source files with your own content
4. **Fonts**: Modify the Google Fonts import at the top of `iDonateStyle.css`
5. **Layout**: Adjust padding and margins in the CSS files

## Notes

- All videos should be in MP4 format for best browser compatibility
- Videos are set to autoplay with sound muted for better user experience
- The design follows modern web standards and best practices
- The website is fully accessible and SEO-friendly
